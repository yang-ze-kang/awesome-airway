什么是肺气道分析What is pulmonary airway analysis

Pulmonary Airway Analysis

气道树是从气管、主/叶/段支气管一路延伸到细支气管的细长、分叉、多尺度管状结构。从胸部 CT 中自动分析气道，是支气管镜导航、经支气管肺结节活检、介入路径规划以及慢阻肺 / 支气管扩张 / 肺纤维化等疾病定量的基础。相比一般器官分割，气道的核心难点在于：类别极度不平衡（远端体素占比极小）、细支气管易断裂、以及对拓扑正确性与连通性的严格要求。 The airway tree is a thin, branching, multi-scale tubular structure running from the trachea and lobar/segmental bronchi down to the distal bronchioles. Automatic airway analysis from chest CT underpins bronchoscopy navigation, transbronchial nodule biopsy, interventional path planning, and quantification of COPD / bronchiectasis / pulmonary fibrosis. Compared with ordinary organ segmentation, its core difficulties are extreme class imbalance (distal voxels are a tiny fraction), easily-broken small bronchi, and strict demands on topological correctness and connectivity.

🧩 主要任务Main tasks

  • 二值气道分割：提取连通的气道掩膜，重点是远端分支保留与拓扑正确。
  • 拓扑 / 连通性建模：中心线/骨架监督、连通性损失、断裂修复。
  • 解剖分支标注：为分支分配标准名称（RB1–10 / LB1–10 等）。
  • 支气管镜导航 / 中心线：路径规划与实时定位。
  • Binary airway segmentation: extract a connected airway mask, prioritizing distal-branch preservation and topology.
  • Topology / connectivity modeling: centerline/skeleton supervision, connectivity losses, break repair.
  • Anatomical branch labeling: assign standard names to branches (RB1–10 / LB1–10, etc.).
  • Bronchoscopy navigation / centerline: path planning and real-time localization.

⚙️ 典型技术脉络Typical technical threads

  • 3D U-Net / nnU-Net 强基线 → 管状敏感损失、类不平衡损失（General Union、clDice）。
  • 连通性感知：体素邻域连通、freeze-and-grow、断裂修复网络。
  • 结构建模：图/树神经网络、Transformer、隐式表示、Mamba/SSM。
  • 标注：沿气道树拓扑传播特征做分支命名。
  • Strong 3D U-Net / nnU-Net baselines → tubular-sensitive & class-imbalance losses (General Union, clDice).
  • Connectivity awareness: voxel-neighbourhood connectivity, freeze-and-grow, break-repair networks.
  • Structural modeling: graph/tree neural nets, Transformers, implicit representations, Mamba/SSM.
  • Labeling: propagate features along the airway-tree topology for branch naming.

气道相关挑战赛Airway challenges

推动气道分析发展的公开挑战赛。ATM 系列（ATM'22 → ATM26）是当前气道树建模的主线基准。Public challenges driving airway analysis forward. The ATM series (ATM'22 → ATM26) is the current mainline benchmark for airway tree modeling.

ATM26 — Airway Tree Modeling 2026

MICCAI 2026

ATM26 将气道分析从 ATM'22 的二值分割推进到结构化气道理解：Track 1 二值气道分割（强调拓扑正确性与远端分支），Track 2 分支级解剖标注（RB1–10 / LB1–10 等）。在真实采集条件（不同分辨率、质量、解剖变异）下评估。ATM26 advances airway analysis from ATM'22's binary segmentation to structured airway understanding: Track 1 binary airway segmentation (emphasizing topology and distal branches) and Track 2 branch-wise anatomical labeling (RB1–10 / LB1–10, etc.), evaluated under realistic acquisition conditions (varying resolution, quality, anatomy).
